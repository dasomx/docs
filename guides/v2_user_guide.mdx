# okchat ver 2.0 사용자 가이드

## 🤖 okchat에 오신 것을 환영합니다! 📖

이 가이드는 okchat v2를 사용하여 자신만의 AI 챗봇을 만들고 관리하는 데 도움을 줄 것입니다.

## 💡 okchat 이해하기

### okchat이란?

okchat은 대규모 언어 모델(LLM)과 사용자의 데이터를 사용하여 AI 기반 챗봇을 쉽게 만들고 배포할 수 있는 플랫폼입니다.

### okchat의 장점

- 고객에게 인간과 유사한 대화 경험 제공.
- 대화 맥락과 사용자 의도를 이해하여 최적의 응답 제공.
- 24/7 다국어 고객 지원 제공.
- 매출 및 고객 만족도 향상.

### 주요 기능

- **사용하기 쉬움**: 초보자도 자신만의 AI 챗봇을 만들 수 있습니다.
- **맞춤 설정 가능**: 자신의 데이터와 대규모 언어 모델을 사용하여 고유한 챗봇을 구축할 수 있습니다.
- **무료로 시작**: okchat은 테스트에 충분한 사용량을 제공하는 무료 플랜을 제공합니다.
- **외부 통합**: KakaoTalk, Naver TalkTalk, LINE, Facebook Messenger, WhatsApp, Instagram 등 인기 있는 메시징 플랫폼과 연결할 수 있습니다.

  [기능설명]

- 웹사이트의 URL을 입력하여 챗봇이 웹사이트의 콘텐츠를 학습하도록 할 수 있습니다. 무료 플랜의 경우 최대 3개의 하위 페이지까지 크롤링이 가능합니다.

  [학습단계]

  - 크롤링할 웹사이트 URL을 지정합니다.
  - 하위 페이지를 크롤링할지 결정합니다.

  [주의사항]

  - 웹사이트 크기에 따른 시간과 크레딧 소비에 주의하세요.

  [기능설명]

  - 직접 제목과 내용을 입력하여 챗봇에게 새로운 지식을 추가할 수 있습니다. 간단한 정보나 FAQ를 추가하기에 적합합니다.

## 🔌 외부 서비스와 통합하기

### API 통합

[API연결단계]

- 1. API 액세스 승인을 받으세요.
- 2. API 기본 설정
     설명: "통합 - 도구" 섹션을 사용하여 API 통합을 생성하고 관리하세요.
     항목 구성:
     - Name: API의 고유 이름을 입력합니다.
     - Description: API의 용도나 기능을 설명합니다.
     - Base URL: API의 기본 URL을 입력합니다.
     - Active: API의 활성화 상태를 설정합니다.

  <img src="https://github.com/HansungJang/Okchat_v2_guide_docs/blob/main/okchat%EC%8B%A4%ED%97%98%EA%B8%B0%EB%A1%9D(~24.12.25.)/3.%20%EC%B1%97%EB%B4%87%20API%20%ED%99%9C%EC%9A%A9/API%2C%20%EC%8B%9C%EC%9E%91%EC%A3%BC%EC%86%8C%20%EC%84%A4%EC%A0%95%ED%95%98%EA%B8%B0.png">

- 3. 엔드포인터 설정
     설명: API의 각 기능에 대한 엔드포인트를 설정합니다. 요청 방식, 경로, 파라미터 등을 정의할 수 있습니다.
     항목구성:
     - Method: HTTP 요청 방식을 선택합니다 (GET, POST 등).
     - Path (URL): 엔드포인트의 경로를 지정합니다.
     - Description: 엔드포인트의 기능을 설명합니다.
     - Query Parameters: URL 쿼리 파라미터를 설정합니다.
     - Path Parameters: URL 경로 파라미터를 설정합니다.

  <img src ="https://github.com/HansungJang/Okchat_v2_guide_docs/blob/main/okchat%EC%8B%A4%ED%97%98%EA%B8%B0%EB%A1%9D(~24.12.25.)/3.%20%EC%B1%97%EB%B4%87%20API%20%ED%99%9C%EC%9A%A9/API%2C%20%EC%9A%94%EC%B2%AD%ED%98%95%EC%8B%9D%20%EC%84%A4%EC%A0%95%ED%95%98%EA%B8%B0.png">

- 4. API 통합관리
     설명: 설정이 완료된 API는 통합 도구 목록에서 확인하고 관리할 수 있습니다.
     항목구성:
     - Edit: API 설정을 수정합니다
     - Delete: API 통합을 삭제합니다
     - Active 상태 확인: API의 활성화 상태를 표시합니다

  <img src ="https://github.com/HansungJang/Okchat_v2_guide_docs/blob/main/okchat%EC%8B%A4%ED%97%98%EA%B8%B0%EB%A1%9D(~24.12.25.)/3.%20%EC%B1%97%EB%B4%87%20API%20%ED%99%9C%EC%9A%A9/API%2C%20%ED%95%99%EC%8A%B5%EB%90%9C%20%EB%8D%B0%EC%9D%B4%ED%84%B0%20%ED%99%95%EC%9D%B8.png">

> 주의사항
>
> - API 통합 기능은 유료 플랜에서만 사용 가능합니다
> - API 호출 시 발생하는 비용은 별도로 청구될 수 있습니다
> - 민감한 인증 정보는 안전하게 보관되어야 합니다
> - API 사용량과 응답 시간을 모니터링하는 것을 권장합니다

### 소셜 미디어 연결

- 원하는 소셜 미디어 비즈니스 채널에 등록하고 승인을 받으세요.
- "통합 - 도구" 섹션을 사용하여 소셜 미디어 연결을 설정하세요.
  - 각 소셜 미디어 플랫폼의 API 키를 입력하세요.
  - URL을 복사하여 소셜 미디어 플랫폼의 웹훅 콘솔에 붙여넣어 웹훅을 구성하세요.

<summary>비즈니스 채널 신청하기</summary>
<div>

   - 사업자 등록 번호<br>

   - 업종별 허가증 (해당되는 경우)<br>
5. 채널 정보와 비즈니스 정보가 정확하게 입력되었는지 확인합니다. 이는 비즈니스 채널 승인에 영향을 줄 수 있습니다.<br>
6. 신청서를 제출합니다. <br>
  
비즈니스 채널 신청에 대한 검토 과정은 약 1주일이 소요됩니다. 비즈니스 채널이 승인되면 카카오톡을 HappyTalk에 연결할 수 있습니다.<br><br>

아래는 okchat Bot 계정 생성부터 KakaoTalk 채널과의 연동까지의 과정을 단계별로 정리한 한글 가이드입니다.

## 1. okchat Bot 계정 생성

1. **okchat Bot 웹사이트**에 접속합니다. (정확한 URL은 명시되지 않았으나, 해당 서비스의 홈페이지를 방문하세요.) <br>
2. **이메일 발송**: `sales@[okchat bot의 URL]` 주소로 아래 정보를 포함하여 메일을 보내주세요. <br>
   - **이름** <br>
   - **휴대폰 번호** <br>
   - **이메일 주소** <br>
3. **계정 승인**: okchat Bot 영업팀으로부터 계정 생성 관련 안내 메일을 기다립니다. <br>

---

## 2. Kakao i Open Builder 라이선스 신청 <br>

1. [**Kakao i 웹사이트**](https://i.kakao.com/)에 접속합니다. <br>
2. **Kakao i Open Builder 라이선스** 신청 페이지에서 관련 정보를 작성합니다. <br>
3. **약 3일 정도** 검토 기간을 기다립니다. (이후 라이선스 승인 여부가 안내됩니다.) <br>

---

## 3. Kakao i Open Builder에서 새 봇 생성 <br>

1. **Kakao i Open Builder** 계정에 라이선스 승인 후 로그인합니다. <br>
2. **“새 봇 만들기”** 버튼(또는 유사 메뉴)을 클릭합니다. <br>
3. **봇 이름 및 기본 설정**을 입력하고 봇을 생성합니다. <br>

---

## 4. KakaoTalk 채널과 봇 연동 <br>

1. **Chatbot Manager Center**(챗봇 관리자 센터)에 접속합니다. <br>
2. 아래 경로로 이동합니다: <br>
   ```
   Chatbot Manager Center
   → 설정(Settings) <>
   → KakaoTalk 채널 연결(KakaoTalk Channel Connection)
   → 운영 채널 연결(Operating Channel Connection)
   ```
3. **연동할 KakaoTalk 채널**을 선택하여 새로 생성된 봇과 연결합니다. <br>

---

## 5. 봇 테스트 <br>

1. **개발자 채널(테스트 채널)**을 설정하여 실제 공개 전 안전하게 테스트를 진행합니다. <br>
2. **테스트 메시지를 주고받으며** 봇의 응답과 기능이 정상적으로 작동하는지 확인합니다. <br>

---

## 6. 봇 관리자 권한 설정 <br>

1. **Chatbot Manager Center**에서 봇 관리자 설정 메뉴로 이동합니다. <br>
2. **관리자로 초대할 사용자**를 추가하거나, 필요한 경우 여러 명에게 권한을 부여합니다. <br>

---

## 7. 최종 확인 및 배포 <br>

- 모든 설정과 테스트를 완료했다면, **KakaoTalk 챗봇**은 **추가 구성 및 배포**가 가능한 상태가 됩니다. <br>

# okchat을 활용한 Kakao i Open Builder | KakaoTalk Chatbot 연동 가이드

## 1. Kakao i Open Builder 연동 준비

1. **Chatbot Manager Center** → **Settings**에서 봇 ID를 확인합니다.
2. 확인한 봇 ID를 복사한 뒤, **[Integration] → [Social Media] → [KaKao] → [API Key]**로 이동하여, 복사한 봇 ID를 붙여넣습니다.
3. 연결 기능 활성화를 위해 슬라이더 바를 클릭하여 **연결(활성화)** 상태로 전환합니다.
4. 만약 **okchat 홈페이지**에 로그인하여 설정을 진행하는 경우, **Kakao Channel Connection**(카카오 채널 연결) 항목에서 해당 봇 ID를 선택해 주세요.

---

## 2. Fallback 스킬 생성 및 연결

1. \***\*[Integration] → [Social Media] → [KaKao] → [Webhook Setup]**에서 **Webhook URL**을 복사합니다.
2. **Chatbot Manager Center** → **Skill** 메뉴로 이동하여 **새 스킬 생성**을 클릭합니다.
3. 새 스킬 이름을 **Fallback**으로 지정하고, 앞서 복사한 **Webhook URL**을 붙여넣은 뒤 저장합니다.
4. **Chatbot Manager Center** → **Scenario** → **Fallback Block**으로 이동하여, **[Parameter Setting]** 드롭다운에서 방금 만든 **Fallback** 스킬을 선택합니다.
5. **[Bot Response] → [Add Response]** 영역에서 **Bot Response Setting**을 **Skill Data**로 설정하고 저장합니다.

---

## 3. Chatbot Start 스킬 생성 및 연결

1. **Fallback 스킬**을 생성했던 방법과 동일하게 진행하되, 스킬 이름을 **Chatbot Start**로 지정합니다.
2. 단, Webhook URL 끝에 **/home**을 추가하여 **(기존 Webhook URL)/home** 형태로 입력해 주세요.
3. **Chatbot Manager Center** → **Scenario**로 이동하여 **[+ Scenario]** 버튼을 클릭합니다.
4. 새 시나리오 블록 이름을 **Chatbot Start**로 입력하고, **[Parameter Setting]** 드롭다운에서 **Chatbot Start** 스킬을 선택합니다.
5. **[Bot Response] → [Add Response]** 영역에서 **Bot Response Setting**을 **Skill Data**로 설정한 뒤 저장합니다.

---

## 4. 카카오 챗봇 시나리오 및 Chatbot Start 버튼 생성

1. **Chatbot Manager Center** → **Scenario** → **Scenario Settings**로 이동하여 **새로운 시나리오**를 생성합니다.
2. **[Bot Generic Menu Setting]** 기능을 활성화하여 **시작 버튼**(스타트 버튼)을 생성합니다.
3. **[+Add Button]**을 클릭하고 버튼 이름을 **Start Chatbot**으로 지정합니다.
4. 해당 버튼을 **Chatbot Start** 블록과 연결한 뒤, 설정을 완료합니다.

---

## 5. 챗봇 배포

1. **Chatbot Manager Center** → **Deploy** 메뉴로 이동하여 **배포(Deploy)** 버튼을 클릭합니다.
2. 위 과정이 완료되면, **Kakao i Open Builder**와 **KakaoTalk 챗봇**이 정상적으로 연동되어 원활하게 동작할 것입니다.

---

이 과정을 모두 마치면, okchat을 통해 Kakao i Open Builder가 연결되어 KakaoTalk 채널에서 챗봇을 사용할 수 있습니다. 필요한 경우 시나리오를 추가하거나, 답변 스킬을 보강해 더욱 풍부한 사용자 경험을 제공해 보세요.
